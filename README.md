# EIPScanner

Implementation of Ethernet/IP scanner in C++

## Features

* Explicit messaging
* Implicit messaging
* File Object (only upload)
* Parameter Object

## Requirements

* CMake 3.5 and higher
* C++14 compiler
* Linux or MacOS

## Installing

```shell script
$ mkdir build && cd build
$ cmake ..
$ cmake --build . --target install
```
